## 🆘 Emergency Checklist

### "The app won't start!"
- [ ] Check TROUBLESHOOTING.md → "Critical Issues"
- [ ] Is MongoDB running? (Check port 27017)
- [ ] Is Node.js installed? (node --version)
- [ ] Did you run npm install? (In both folders)
- [ ] Do you have .env file? (With all variables)

### "Everything's broken!"
- [ ] Try full reset in TROUBLESHOOTING.md → "Complete Reset"
- [ ] Delete node_modules, reinstall
- [ ] Clear browser cache (Ctrl+Shift+Delete)
- [ ] Restart all services
- [ ] Try again from scratch

### "Still broken!"
1. Read error message carefully
2. Search error in TROUBLESHOOTING.md
3. Check browser console (F12) for details
4. Check backend terminal output
5. Check .env file has all variables
6. Try the recovery steps in TROUBLESHOOTING.md
